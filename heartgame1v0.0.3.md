# HeartGame1 游戏文档

## 1. 安装指南

### 1.1 Windows 安装指南

#### 系统要求
- **操作系统**：Windows 7 或更高版本
- **处理器**：1 GHz 或更快的处理器
- **内存**：至少 2 GB RAM
- **存储空间**：至少 100 MB 可用空间
- **显卡**：支持 DirectX 9.0c 或更高版本
- **其他**：需要安装 Visual C++ 运行库

#### 下载链接
- **64位版本**：`heartgame1.exe`
- **32位版本**：`heartgame1(32bit).exe`

#### 安装过程
1. **下载安装程序**：从官方网站或指定来源下载适合您系统架构的安装程序
2. **运行安装程序**：双击下载的 `.exe` 文件开始安装
3. **跟随安装向导**：
   - 选择安装语言（支持中文）
   - 阅读并接受许可协议
   - 选择安装目录（默认：`C:\Program Files\heartgame1`）
   - 选择开始菜单文件夹
   - 点击「安装」按钮开始安装
4. **完成安装**：安装完成后，点击「完成」按钮退出安装向导

#### 环境配置
- 安装程序会自动配置必要的环境变量
- 确保您的系统已安装最新的 DirectX 运行库
- 对于 Windows 7 用户，建议安装 Service Pack 1

#### 验证安装成功
1. **启动游戏**：从开始菜单或桌面快捷方式启动游戏
2. **检查游戏窗口**：游戏应正常启动并显示主界面
3. **验证 LICENSE.txt**：游戏启动时会自动检查 LICENSE.txt 文件是否存在
4. **测试基本功能**：尝试进行几次攻击操作，确保游戏机制正常工作

### 1.2 Linux 安装指南

#### 系统要求
- **操作系统**：Ubuntu 18.04 或更高版本，其他 Debian 衍生版
- **处理器**：1 GHz 或更快的处理器
- **内存**：至少 2 GB RAM
- **存储空间**：至少 100 MB 可用空间
- **显卡**：支持 OpenGL 2.1 或更高版本
- **其他**：需要安装 Python 3 和 Pygame

#### 下载链接
- **Debian 包**：`heartgame1_0.0.3_all.deb`

#### 安装过程
1. **下载 Debian 包**：从官方网站或指定来源下载 `.deb` 包
2. **安装依赖**：
   ```bash
   sudo apt update
   sudo apt install python3 python3-pygame
   ```
3. **安装游戏**：
   ```bash
   sudo dpkg -i heartgame1_0.0.3_all.deb
   ```
4. **解决依赖问题**（如果需要）：
   ```bash
   sudo apt install -f
   ```

#### 环境配置
- 确保您的系统已安装必要的音频和视频驱动
- 对于使用 Wayland 的系统，可能需要切换到 Xorg 以获得更好的兼容性

#### 验证安装成功
1. **启动游戏**：
   ```bash
   python3 /usr/games/main.py
   ```
2. **检查游戏启动**：游戏应正常启动并显示主界面
3. **验证 LICENSE.txt**：游戏启动时会自动检查 LICENSE.txt 文件是否存在
4. **测试基本功能**：尝试进行几次攻击操作，确保游戏机制正常工作

## 2. 游戏玩法

### 2.1 核心机制

#### 攻击系统
- **攻击次数**：每次攻击会增加攻击计数
- **攻击阶段**：
  - 攻击次数 1-3 次：第一阶段
  - 攻击次数 4-7 次：第二阶段
  - 攻击次数 8-9 次：第三阶段
  - 攻击次数 10 次：特殊攻击动画和结局场景
- **暴力指数**：每攻击 5 次，暴力指数（LV）增加 1

#### 等级系统
- **生命值**：LV1 时为 20，每提升 1 级增加 15 点
- **伤害减免**：LV1 时减少 50%，之后每提升 1 级调整为 -50%+5%*(LV-1)
- **饶恕冷却时间**：LV1 时减少 30%，之后每提升 1 级调整为 -30%+2%*(LV-1)
- **等级上限**：LV20

#### 特殊机制
- **10th 攻击**：完成第 10 次攻击后，会播放特殊攻击动画序列，并显示结局场景
- **伪善行为**：当暴力指数较低但选择攻击行为时，系统会判定为伪善行为，可能触发相应的游戏反馈或结局影响

### 2.2 操作方式

#### 基本操作
- **攻击**：点击攻击按钮或按对应快捷键
- **饶恕**：点击饶恕按钮或按对应快捷键
- **移动**：使用方向键或鼠标控制角色移动
- **躲避**：在攻击阶段使用方向键或鼠标躲避攻击

#### 界面元素
- **生命值**：显示当前角色的生命值
- **等级**：显示当前暴力指数（LV）
- **攻击计数**：显示已进行的攻击次数
- **攻击阶段**：显示当前所处的攻击阶段
- **游戏提示**：显示游戏操作提示和剧情信息

### 2.3 游戏目标

- **主要目标**：完成所有攻击阶段，体验完整的游戏剧情
- **挑战目标**：在不同等级下尝试不同的游戏策略
- **收集目标**：体验不同的攻击动画和结局场景

### 2.4 关卡特点

- **第一阶段**（攻击次数 1-3）：基础攻击模式，难度较低
- **第二阶段**（攻击次数 4-7）：攻击模式变得更加复杂，需要更高的反应速度
- **第三阶段**（攻击次数 8-9）：高难度攻击模式，需要精准的操作
- **特殊阶段**（攻击次数 10）：播放特殊攻击动画，展示结局场景

## 3. 注意事项

### 3.1 Windows 平台注意事项

#### 兼容性问题
- **Windows 7**：需要安装 Service Pack 1 和最新的补丁
- **Windows 10/11**：可能需要以管理员身份运行游戏
- **32位系统**：请使用 32位版本的安装程序

#### 性能优化建议
- **降低游戏分辨率**：如果游戏运行卡顿，可以尝试降低游戏分辨率
- **关闭后台程序**：运行游戏时关闭不必要的后台程序
- **更新显卡驱动**：确保您的显卡驱动是最新版本

#### 常见错误及解决方法
- **LICENSE.txt 文件不存在**：
  - 解决方法：确保游戏安装目录中存在 LICENSE.txt 文件
- **无法加载音效**：
  - 解决方法：检查音频驱动是否正常，确保音箱或耳机已连接
- **游戏崩溃**：
  - 解决方法：以管理员身份运行游戏，或尝试兼容模式运行

#### 数据备份策略
- **配置文件**：定期备份游戏目录中的配置文件
- **游戏进度**：游戏会自动保存进度，建议定期备份游戏目录

#### 系统安全相关提示
- **下载来源**：只从官方网站或可信来源下载游戏
- **防病毒软件**：如果防病毒软件误报，请将游戏添加到白名单
- **管理员权限**：只在必要时以管理员身份运行游戏

### 3.2 Linux 平台注意事项

#### 兼容性问题
- **Wayland**：可能存在兼容性问题，建议切换到 Xorg
- **不同发行版**：虽然游戏设计为兼容所有 Debian 衍生版，但在某些发行版上可能需要额外配置

#### 性能优化建议
- **关闭不必要的服务**：运行游戏时关闭不必要的系统服务
- **调整系统资源**：根据系统性能调整游戏设置
- **使用硬件加速**：确保系统已启用硬件加速

#### 常见错误及解决方法
- **LICENSE.txt 文件不存在**：
  - 解决方法：游戏会自动检查 `/usr/share/doc/heartgame1/LICENSE.txt.gz` 文件
- **无法加载音效**：
  - 解决方法：安装必要的音频库，如 `libasound2`
- **Pygame 初始化失败**：
  - 解决方法：确保已正确安装 Pygame，运行 `pip3 install pygame`

#### 数据备份策略
- **配置文件**：定期备份 `/usr/share/heartgame1/` 目录
- **游戏进度**：游戏会自动保存进度，建议定期备份相关文件

#### 系统安全相关提示
- **安装来源**：只从官方网站或可信来源下载 `.deb` 包
- **权限管理**：避免以 root 用户运行游戏
- **依赖检查**：安装前检查依赖项，确保系统安全
